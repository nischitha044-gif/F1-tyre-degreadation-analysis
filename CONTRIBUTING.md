# Contributing Guidelines

Thanks for your interest in contributing to this tyre degradation analysis project!

## Ways to contribute
- Add or improve data ingestion (with provenance and license)
- Improve cleaning or EDA code
- Add new analyses or visualizations
- Add tests or CI workflows
- Fix bugs or refine documentation

## Getting started
1. Fork the repo and create a branch for your change
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Make changes and run notebooks/scripts to verify results
4. Open a pull request with a clear description of your changes

## Data & Licensing
If you add or change data sources, please document the source, URL, and any licensing restrictions in the README.MD.

## Communication
Open issues for larger changes or if you want to discuss an idea before implementing it.
